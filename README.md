# UserApi

A .NET 7 Web API with MongoDB CRUD operations for users.

## Features
- Create, Read, Update, Delete users
- MongoDB integration
- Swagger UI for testing endpoints

## Running
1. Update `appsettings.json` with your MongoDB connection string.
2. Run `dotnet run`
3. Visit `http://localhost:5000/swagger`
